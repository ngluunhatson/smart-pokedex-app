# Smart Pokédex

A Next.js project that brings the world of Pokémon to your fingertips.

## About

This project is a smart Pokédex, a digital encyclopedia of Pokémon. It uses the latest web technologies to provide a fast, responsive, and interactive experience for Pokémon fans.

## Features

* Browse a vast collection of Pokémon, including their names, images, and forms
* Filter and sort Pokémon by various criteria, such as type, generation, and name
* View detailed information about each Pokémon, including its stats, moves, and habitats
* Use the built-in theme picker to customize the app's appearance to your liking

## Getting Started

To get started with the project, follow these steps:

1. Clone the repository: `git clone https://github.com/ngluunhatson/smart-pokedex-app.git`
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev` 
4. Open your browser and navigate to `http://localhost:3000`

## Technologies Used

* Next.js: A popular React-based framework for building server-rendered and statically generated websites and applications.
* Convex: A cloud-based backend platform that provides a scalable and secure way to store and manage data.
* Clerk: A cloud-based authentication service that provides a simple and secure way to manage user authentication and metadata.
* Tailwind CSS: A utility-first CSS framework for building custom user interfaces.
* Storybook: A popular tool for building and testing UI components in isolation.
